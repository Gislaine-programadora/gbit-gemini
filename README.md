# gbit-gemini - Frontend

Frontend desenvolvido com Vite e TypeScript para o projeto gbit-gemini.

## 🚀 Tecnologias

- React
- Vite
- TypeScript
- Tailwind CSS

## 📦 Instalação

```bash
npm install
```

## 🏃 Executar

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 🎨 Tema

Tema atual: **Gbit Dark**
